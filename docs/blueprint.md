# **App Name**: MitraAI: Your Relationship Guide

## Core Features:

- Conversation Coaching: Interactive chat interface providing context-specific advice, conversation starters, and tips for engaging dialogue.
- Approach Guide: Step-by-step guided module with confidence building exercises, body language tips, and conversation starters for first encounters.
- Safe Communication Analyzer: AI feature providing real-time feedback on tone, appropriateness, and potential misunderstandings in user's intended messages. Tool will learn red flags and context for safer, effective communication.
- Interest Detection Q&A: Interactive questionnaire system analyzing user interactions and providing a percentage-based interest level indicator with specific behavioral examples.
- Maturity Enhancement: Personalized advice system providing suggestions for communication style improvements, emotional intelligence, conversation depth, and personal growth.
- User Authentication: Firebase Authentication to ensure only logged in users access personal relationship advice and track progress.
- Persistent Chat History: Conversation history saved for continuity.

## Style Guidelines:

- Primary color: Gentle sky blue (#87CEEB) to evoke calmness and approachability.
- Background color: Very light blue (#F0F8FF), almost white, providing a clean and unobtrusive backdrop.
- Accent color: Soft lavender (#E6E6FA) for highlights and CTAs, adding a touch of sophistication without being overwhelming.
- Body and headline font: 'PT Sans' for a modern look, combined with some warmth.
- Clean and modern icons for bottom navigation and action cards, enhancing user experience.
- Mobile-first responsive design, with a clean, modern layout and calming color schemes. The welcome screen introduces the app's purpose, with a main dashboard showing clear paths 'How to Talk,' 'Approach Guide,' 'Check Interest Level,' and 'Become More Mature'. Each selection flows seamlessly into its designated area.
- Subtle transitions and animations to enhance user engagement and provide feedback.
